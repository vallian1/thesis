# 长缺测强度实验（片段级 fCEDAS）

## 实验设定
- 数据：2025.3.3 85个数据汇总_标签.xlsx
- 模拟方式：按真实车辆标签序列注入随机漏检，同时控制总体漏检率与最大连续缺测长度；上游短缺测容忍阈值设为 2，即连续缺测长度大于 2 时产生轨迹片段。
- 评价方式：以观测点为单位，比较重构前（每个片段单独成类）与重构后（fCEDAS 聚类结果）的 FMI/ARI；同时统计长度分布变化。

## 边界判断
- 在当前数据上，当总体漏检率不高于 0.55 且最大连续缺测长度不超过 6 时，方法仍保持较强的片段拼接能力；重构后 FMI 保持在 0.924 以上，中位长度增益约为 2.2–3.0 倍。
- 当总体漏检率升高到 0.65 时，性能开始明显下降。特别是在最大连续缺测长度由 3 增加到 4 后，重构后 FMI 由 0.893 下降到 0.866，簇数明显偏离真实车辆数，说明方法进入明显失稳区间。
- 因此，从本组实验看，总体漏检率约 0.65 是主要失效拐点；在这一强度下，最大连续缺测长度达到 4 可视为明显失效的起始点。
